A changelog file is not necessary but it helps you and others keep track of the changes you make version to version.
You usually would want to sort it into however many categories you want to make changes easier to find.
This won't show up in game and is just seen by those who open the files.

Ex.

Version 1.3

-----------------------------------------------------------------------------------------

New content

-Added suit: Superman (Comics)
-Added suit: Batman (Comics)

-Added alt: Superman (Comics) New 52

Bug fixes

-Fixed Spider-Man (Comics) crashing game when activating webshooters underwater.
-Fixed an issue that caused Captain America (Comics) to play the wrong sound effect when getting hit.

Balancing

-Increased Spider-Man (Comics) punch damage from 4.5 to 8.0
